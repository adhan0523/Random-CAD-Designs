# 🌪️ Fan Impeller Design in SolidWorks

## Overview

This repository contains the CAD design of a fan impeller created in SolidWorks. The project was developed to practice advanced CAD modeling techniques and to gain a better understanding of the geometry and design principles used in rotating airflow components.

The model consists of multiple blades arranged around a central hub, forming an impeller capable of generating airflow when rotated. This project serves as a practical exercise in mechanical design and 3D modeling.

---

## Project Objectives

* Design a realistic fan impeller model
* Improve SolidWorks part modeling skills
* Practice surface and loft-based design techniques
* Understand the fundamentals of impeller geometry
* Develop a portfolio-ready CAD project

## Design Features

* Multi-blade impeller design
