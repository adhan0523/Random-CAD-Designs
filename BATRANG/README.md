# 🦇 Batman Logo Design in SolidWorks


---

## Overview

This repository contains a 3D model of the iconic **Batman logo** designed in SolidWorks.

The project was created as a CAD practice exercise to improve sketching accuracy, feature-based modeling, and overall proficiency in SolidWorks. Although the model is simple compared to complex mechanical assemblies, it provided a great opportunity to practice creating clean geometry from a recognizable design.

---

## Why I Built This Project

While learning SolidWorks, I wanted to work on projects beyond mechanical components to improve my sketching and modeling skills.

The Batman logo was a fun challenge because it required careful attention to symmetry, curves, and proportions while maintaining a clean and accurate design.

---

## Project Objectives

- Recreate the Batman logo in SolidWorks.
- Practice 2D sketching and feature-based modeling.
- Improve precision in geometric design.
- Strengthen CAD modeling fundamentals.
