# Helical Bevel Gear Design

## Overview

This repository contains the CAD model of a Helical Bevel Gear created in SolidWorks as part of my CAD learning journey. The project focuses on designing a gear that combines the smooth operation of helical teeth with the angular power transmission capability of bevel gears.

Through this design, I explored advanced sketching, feature creation, and geometric modeling techniques commonly used in mechanical power transmission systems.

## Design Features

* 3D modeled in SolidWorks
* Helical tooth profile
* Bevel gear geometry
* Detailed mechanical design
* Suitable for angular power transmission applications
* Parametric CAD model

## Applications

Helical bevel gears are commonly used in:

* Automotive transmissions
* Industrial gearboxes
* Conveyor systems
* Power transmission equipment
* Machine tools
* Heavy machinery
